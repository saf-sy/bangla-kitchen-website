# Bangla Kitchen & Sweets — Website

Next.js marketing site for Bangla Kitchen & Sweets (Sugar Land, TX): hero, story, menu, delivery, catering, and contact.

## Stack

- [Next.js](https://nextjs.org/) 15 (App Router)
- React 19
- Tailwind CSS 3
- Framer Motion

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Optional assets

- **Lo-fi audio:** add `public/vintage-dhaka.mp3` for the optional footer-side player (the UI still works if the file is missing).
